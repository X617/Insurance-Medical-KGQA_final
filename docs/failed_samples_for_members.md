# 失败样例记录（prompt / 检索调优）

在前端或跑批脚本手工/自动联调时填写。  
请把**可复现**信息写全，便于针对 `query_understanding.py`、`graph_retriever.py`、`rag_engine.py` 等做迭代。

## 回归样例来源

- 样例列表：`tests/data/week3_regression_cases.json`（单轮与多轮）
- 自动跑批：`python scripts/week3_regression_runner.py --base-url http://127.0.0.1:8000 --out-csv import_logs/week3_regression_run.csv`

|------|---------|------|----------|-------------------------|----------------|----------|-----------------------------|
| | | | | | | | |

## 典型应记录的现象类型（枚举）

- **年龄合规**：推荐了超龄产品，或混淆「投保年龄」与「最高续保年龄」
- **险种错配**：问重疾险却主推医疗险/养老
- **多轮指代丢失**：追问「上面的」仍大量引入新检索或新产品
- **地域/预算**：养老院城市、价格筛选与图谱字段不一致导致空结果
- **幻觉**：Context 为空或很短时仍能编造具体产品名/条款细节
